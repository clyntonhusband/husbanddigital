How to Use

1. Include the script:

<script src="shared-header.js"></script>

2. Add the container (optional - will be auto-created):

<div id="shared-header"></div>

3. Initialize with your page ID and title:

const user = initSharedHeader('mypage', 'My Page Title');

4. Access user info anytime:

const user = window.getSharedUser();
console.log(user.name, user.email);

Configuration

Edit the top of shared-header.js to customize:

// Navigation links
const NAV_LINKS = [
  { href: 'index.html', label: 'Home', id: 'index' },
  { href: 'dashboard.html', label: 'Dashboard', id: 'dashboard' },
  { href: 'admin_tools.html', label: 'Admin', id: 'admin', restricted: true }
];

// Admin users (by name)
const ADMIN_USERS = ['Admin User'];

// Project name in header
const PROJECT_NAME = 'My Project';

Events

Listen for user login events:

window.addEventListener('userLogin', (e) => {
  console.log('User logged in:', e.detail.name, e.detail.email);
  // Update your UI here
});
